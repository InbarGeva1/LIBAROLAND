# 🌴 LIBAROLAND

> Libar × Tomorrowland × Thailand 2026

LIBAROLAND is a private travel application built specifically for our Thailand trip in December 2026.

The goal is to replace multiple travel apps with one beautiful, intuitive experience.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- Firebase
- React Router
- GitHub
- Google Maps API
- Google Speech API
- Exchange Rate API
